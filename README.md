
# starrylang

A JavaScript implementation of Starry (https://esolangs.org/wiki/Starry), a stack-based, Turing complete, esoteric programming language.

## installation

```
npm install starrylang
```

## usage

```
const Starry = require('starrylang');
// prints hello world
Starry(`            +               +  *       +    
 * + .        +              +  *       +   
  *     * + .            +     * + . + .    
    +     * + .              +            + 
 *         +     * * + .                 + *
 + .              + +  *           +     *  
   * + .             + * + .        +     * 
+ .           + * + .             + * + .   
           +            +  *         +     *
 * + .`);
 ```
